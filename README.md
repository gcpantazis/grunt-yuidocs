# grunt-yuidocs

Custom Grunt task that generates YUIDocs.

##How to use

* Install with: ```npm install grunt-yuidocs```.
* Call ```grunt.loadNpmTasks('grunt-yuidocs')``` in grunt.js

* In your grunt.js config:

	```
	yuidocs: {
		js: {
			sourcePath: /path/to/js
			outputPath: /path/to/docs
		}
	}
	```
